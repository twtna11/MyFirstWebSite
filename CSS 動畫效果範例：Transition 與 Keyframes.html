CSS 動畫範例總覽

1. 轉場 (Transition)：懸停按鈕變色

**作用：** 處理「開始」到「結束」的單一狀態平滑過渡，需事件觸發（如 `:hover`）。
**核心 CSS：** `transition: background-color 0.3s ease;`

2. 關鍵影格 (Keyframes)：連續旋轉載入圖示

**作用：** 用於複雜或循環動畫，可自動播放，不需要事件觸發。
**核心 CSS：** `@keyframes spin {...}` 和 `animation: spin 1s linear infinite;`

3. 關鍵影格 (Keyframes)：脈衝/彈跳效果 (多段狀態)

**作用：** 通過多個 `0%`, `50%`, `100%` 定義複雜的動畫序列。
**核心 CSS：** `@keyframes pulse {...}` 和 `animation: pulse 2s ease-in-out infinite alternate;`
